SUÍTE DE PROGRAMAS DE ESCRITÓRIO

Chama-se suíte de programas, ou pacotes de programas, um conjunto de softwares comercializados juntos, em uma mesma embalagem. Os principais produtos desta categoria são desenvolvidos para trabalho de escritório(programas para uso geral). Exemplos de suíte de programas de escritório: LibreOffice, Microsoft office, Google Docs e Apple iWork.

1.Microsoft Office

É a suíte da empresa Microsoft, a mesma que desenvolve o sistema operacional windows. Fazem parte do Microsoft Office:
Microsoft Word: processador de texto;
Microsoft Excel: planilha eletrônica;
Microsoft PowerPoint: programa para criação de apresentações multimídia, ou seja, Trata-se de um software que permite realizar apresentações através de “slides”.

2.LibreOffice

O LibreOffice é uma potente suíte de escritório. É uma suíte de escritório livre e de código aberto, Ele surgiu de uma ramificação do OpenOffice.org em 2010. Fazem parte do LibreOffice:
LibreOffice Writer: processador de texto;
LibreOffice Calc: planilha eletrônica;
LibreOffice Impress: apresentações de slides.
